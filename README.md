
# RewardsAscended

A rewards system for Ark: Survival Ascended.

Features:

- configurable random or guaranteed rewards
- console and RCON commands for giving rewards to players
- API for developers to use with other plugins

Console & RCON Commands:

- RA.Reload (Reload the config)
- RA.Reward <eos_id> <reward_id_from_config> (Gives everything configured for a reward to the player)
